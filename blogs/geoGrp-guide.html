geoGrp Guide

Sarkuya, 2019-06-16

Basic Usage

Your browser does not support HTML5 Canvas. 
var canvas = document.getElementById("canvas1");

geoGrp.setupCanvas(canvas);
geoGrp.setupCoordinate("up", geoGrp.getCenterPoint(), 1, 2, 1, 2, true);

geoGrp.ctx.fillStyle = "hsla(150, 50%, 50%, 0.2)";
geoGrp.fillRoundedRect(Rect(-0.4, 0.4, 0.8, 0.8), 0.05);

Since geoGrp supports multiple canvas in one page, you can build a big library of many vector images. Save each of them in Firefox or Chrome if needed.

Your browser does not support HTML5 Canvas.

Codes:
