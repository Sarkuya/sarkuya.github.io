此时，有以下公式：

sin(β) = y / r;
cos(β) = x / r;
tan(β) = y / x;
cot(β) = x / y;
sec(β) = r / x;
csc(β) = r / y;

这些公式与第一象限中的公式完全一样。不一样的是，由于在第二象限中，β的值变成180o - θ = 180o - 120o = 60o. 且x的值变成了负数。

现在来看，β与落在第一象限的角度为60o的三角函数的情况。

Your browser does not support HTML5 Canvas.

可以看出：

sin(120o) = y / r = y' / r' = sin(60o)
cos(120o) = x / r = -x' / r' = -cos(60o)
tan(120o) = y / x = y' / -x = -tan(60o)
cot(120o) = x / y = -x / y' = -cot(60o)
sec(120o) = r / x = r' / -x' = -sec(60o)
csc(120o) = r / y = r' / y' = csc(60o);

也即，

sin(180o - θ) = sin(θ)
cos(180o - θ) = -cos(θ)
tan(180o - θ) = -tan(θ)
cot(180o - θ) = -cot(θ)
sec(180o - θ) = -sec(θ)
csc(180o - θ) = csc(θ);

同理，在第三象限中，

sin(180o + θ) = -sin(θ)
cos(180o + θ) = -cos(θ)
tan(180o + θ) = tan(θ)
cot(180o + θ) = cot(θ)
sec(180o + θ) = -sec(θ)
csc(180o + θ) = -csc(θ);

同理，在第四象限中，

sin(360o - θ) = -sin(θ)
cos(360o - θ) = cos(θ)
tan(360o - θ) = -tan(θ)
cot(360o - θ) = -cot(θ)
sec(360o - θ) = sec(θ)
csc(360o - θ) = -csc(θ);

可见，当角度在0至360度的范围的变化时，三角函数总能自动地反映出坐标值的正负值。

根据夹角求坐标值

三角函数可以转换为以下形式：


    
        x=r×cosθ
    



    
        y=r×sinθ

这表示，当知道半径r, 以及角度θ, 我们总能求出在坐标系中的坐标值。

这一点有2个主要应用：

  1. 将圆弧等分为n段，则圆弧上每个节点可确定
  2. 将某线段围绕原点旋转特定角度，则旋转后的坐标值可确定
