NetBeans的安装

NetBeans的最新版本已经更新为Apache NetBeans。

安装JDK

在Mac OS X下，有".dmg"的安装包，可以直接安装。只要JDK的版本大于1.8.0就可以安装Apache NetBeans IDE 11.0。

验证：

java -version

安装Ant

下载Ant.

将所下载的Ant顶层文件夹复制到"/usr/local"。

在终端中输入：

export PATH=/usr/local/apache-ant-1.10.5/bin:$PATH
echo 'export PATH=/usr/local/apache-ant-1.10.5/bin:$PATH' >> ~/.profile

第一行代码为当前session设置路径，第二行代码为当前用户的以后所有session设置路径。

验证：

ant -version

安装NetBeans

NetBeans的安装有源包(source)及运行包(binaries)。这里介绍运行包的安装。

下载运行包，解压，并复制到"/Applications"文件夹中。

将netbeans/bin下面的"netbeans"复制到桌面上，会自动创建一个替身(Windows操作系统下称为快捷方式)。双击这个替身即可运行NetBeans。

配置

设置默认的编码为UTF-8

在"netbeans/etc"文件夹下，将"netBeans.conf"拖至NetBeans中编辑。

设置netbeans-default_options的最前面的内容为：

netbeans_default_options=“-J-Dfile.encoding=UTF-8 ...”

默认的文件浏览器

对于类似于SVG的文件，按右键，有一个"View"菜单。如果希望使用Safari打开，选Preferences -> General -> Web Browser -> Safari.
